# Banking-System-Repository
A professional Banking Management System built with **ASP.NET Core Web API** 

## Author
**Omar Mahmoud Sanad**


## Technologies
- ASP .Net Core Web API
- Entity Framework Core (Code First)
- SQL Server
- Linq(Language Integrated Query)
- JWT Authentication
- AutoMapper
- BCrypt For Password Hashing
- Swagger
- Global Exception Handling


## Features
- Customer Management
- Employee Management
- Branch Management
- Account Management
- JWT Authentication
- Authorization
- Filtering & Searching
- Pagination
- Sorting
- Data Seeding
- Fluent API Configurations

## Architecture
- Onion Architecture :
                      - Domain Layer (Core)
                      - Repository Layer
                      - Service Layer
                      - API Layer
  -DTOs

## How To Run
1. Clone Repository
2. Update Connection String
3. Run Migration
4. Run Project
